Софт эмулирует реального пользователя в приложении Farcaster.

Что реализовано:

Установка апк файлов.

Использование прокси с помощью SuperProxy.

Свапы токенов (USDC, ETH) внутри приложения.

Достает приваткеи внутри настроек и сохраняет рядос с сид фразой аккаунта.



Скачивание и установка:

git clone https://github.com/FreEZfT/Farcaster-Mobile-Emulator-

cd FarLD

python -m venv .venv

.venv\Scripts\activate

pip install -r requirements.txt

python LDCaster.py
